⚙️ Instrucciones de conexión — sigue estos pasos una sola vez:
  1. Abre Google Sheets → crea una hoja nueva.
  2. En el menú: Extensiones → Apps Script.
  3. Borra el código existente y pega el contenido del archivo apps_script_v2.gs.
  4. Clic en Implementar → Nueva implementación → Tipo: Aplicación web → Acceso: Cualquier persona → Implementar.
  5. Acepta los permisos que pide Google.
  6. Copia la URL que aparece y pégala abajo.
  7. Importante: Abre este archivo HTML desde Google Drive (súbelo y ábrelo desde ahí), no con doble clic.
